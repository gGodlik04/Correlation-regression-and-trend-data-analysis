Заданы два класса, для которых в файле dataset_1.txt заданы n1 и n2 значений фактора х соответственно. Цель: по обучающей выборке составить модель определения принадлежности по новому значению фактора объекта к одному из заданных классов.

Для одного из приведенных датасетов провести дискриминантный анализ:

1. Проверить нормальность распределения фактора для каждого класса, построить гистограммы распределения фактора для каждого класса.
2. Найти априорные вероятности для каждого класса.
3.Найти оценки параметров: средние значения и среднеквадратические отклонения для каждого класса. Убедиться в справедливости гипотезы о равенстве дисперсий. В качестве общего ср.кв.отклонения взять их среднее арифметическое.
4. Построить дискриминантную функцию (пользовательскую)
5. Провести прогноз для новых значений фактора (определить принадлежность к 1 или 2 классу).
