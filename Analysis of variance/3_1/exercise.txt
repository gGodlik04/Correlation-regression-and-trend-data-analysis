В файле anova_one_way.txt представлены данные регистрации объемов продаж (sales) в зависимости от способов рекламы товара (type). Считать данные, перевести type в факторный вид.
1. Построить Boxplot для визуализации данных зависимости продаж от способов рекламы.
2. Проверить с помощью теста Шапиро-Уилка нормальность распределения.
3.Проверить с помощью теста Бартлетта гомогенность дисперсий распределения.
4. Провести провести дисперсионный анализ aov(). По сводке результатов summary сделать вывод о принятии или опровержении нулевой гипотезы. В случае принятия альтернативной гипотезы провести множественное сравнение с помощью теста Тьюки, обосновать различия по summary.
